üîß Syntax (Structure)

    return value;
}

Breakdown:

  • returnType : Fn ka return type, jo return kia jayega (e.g., int, float)
  • functionName() : Fn ka naam + empty brackets (no i/p
  • { } : Fn ka kaam yha likha jata h
  • return value; : Fn ek value return karta h
